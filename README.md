# ポートフォリオサイト

シンプルなHTML/CSS/JavaScriptで作成されたポートフォリオサイトです。

## ファイル構成

- `index.html` - メインHTMLファイル
- `styles.css` - スタイルシート（ダークモード対応）
- `script.js` - JavaScript（テーマ切り替え、スムーズスクロール）
- `assets/images/` - 画像ファイル用フォルダ

## 使い方

1. `index.html`をブラウザで開く
2. プロフィール画像を`assets/images/profile.jpg`に配置してください
3. HTMLファイル内の以下の部分を編集して、あなたの情報に置き換えてください：
   - Aboutセクション：名前、自己紹介文
   - Skillsセクション：スキルリスト
   - Experienceセクション：経歴情報

## 機能

- ダークモード/ライトモードの切り替え（localStorageで保存）
- スムーズスクロール
- レスポンシブデザイン（モバイル対応）

## カスタマイズ

- 色の変更：`styles.css`のCSS変数を編集
- コンテンツの追加：`index.html`の各セクションを編集
- スタイルの調整：`styles.css`を編集
